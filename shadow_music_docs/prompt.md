# Build Instructions

Build a complete full-fledged premium music streaming application inspired by Spotify.

This must be production-grade.

---

# Stack

Frontend:
- Next.js App Router
- TypeScript
- TailwindCSS
- Zustand
- Framer Motion

Backend:
- FastAPI

Authentication:
- Firebase Authentication

Search:
- Meilisearch

Database:
- Firestore

Storage:
- Google Drive

# UI Fidelity Requirements

The application UI must resemble Spotify as closely as technically possible.

Reference:
https://open.spotify.com/

Replicate:
- layout structure
- spacing system
- sidebar proportions
- player positioning
- hover behaviors
- gradients
- card sizing
- typography hierarchy
- navigation patterns
- animations
- transitions
- music player interactions
- playlist page layouts
- search experience
- library organization
- scrollbar styling
- album card interactions
- sticky player behavior
- responsive breakpoints
- desktop experience
- immersive dark appearance

Requirements:
- pixel-level polish
- smooth animations
- premium-quality transitions
- no generic dashboard styling
- no simplified UI
